# Love Calculator API

## Request

BASE_URL: `https://localhost:3000/api/v1`

Query: `/calculate?personA=Marry&personB=James`

```
<!-- Example URL -->
https://localhost:3000/api/v1/calculate?personA=Marry&personB=James

```

## Response

```json
{
  "result": 96,
  "message": "Marry and James, you both are perfect couple."
}
```
